# Asbemord

این مخزن حاوی وب‌سایت استاتیک روستای اسبمرد است. پروژه با هدف معرفی روستا، نمایش تصاویر و فراهم کردن راه‌های ارتباطی برای بازدیدکنندگان طراحی شده است.

## نمای کلی

این پروژه یک صفحه‌ی ساده و مدرن برای نمایش اطلاعات روستا است که شامل:

- اسلایدشو با تصاویر مختلف
- طراحی مینیمال و تمیز
- اطلاعات تماس و لینک‌های ارتباطی
- پشتیبانی از استایل و افکت‌های ساده با HTML، CSS و JavaScript

## ویژگی‌ها

- نمایش اسلایدشو با چند تصویر
- طراحی Responsive و مناسب برای مرورگرهای مختلف
- استفاده از Nivo Slider برای تجربه‌ی بهتر
- بدون نیاز به نصب وابستگی یا Build
- آماده برای انتشار روی GitHub Pages

## ساختار پروژه

- index.html: فایل اصلی صفحه‌ی وب
- assets/css: فایل‌های استایل پروژه
- assets/js: اسکریپت‌ها و کتابخانه‌های JavaScript
- image: تصاویر استفاده‌شده در اسلایدشو

## راه‌اندازی محلی

برای مشاهده‌ی پروژه در محیط محلی، فایل زیر را در مرورگر باز کنید:

```bash
index.html
```

یا با یک سرور محلی:

```bash
python -m http.server 8000
```

سپس به نشانی زیر بروید:

```text
http://localhost:8000
```

## انتشار روی GitHub Pages

این پروژه به صورت استاتیک است و می‌تواند مستقیم روی GitHub Pages منتشر شود.

### نکته
- برای استفاده با دامنه سفارشی، فایل CNAME در ریشه پروژه قرار داده شده است.
- اگر از GitHub Pages استفاده می‌کنید، در تنظیمات مخزن گزینه‌ی Pages را فعال کنید.
